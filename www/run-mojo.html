glassfish:run

Full name:

org.glassfish:maven-glassfish-plugin:1.0-alpha-4:run

Description:

Executes GlassFish v3 inside the current Maven and deploys the application being developed.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Requires dependency resolution of artifacts in scope: runtime.
  • Invokes the execution of the lifecycle phase compile prior to executing itself.

Optional Parameters

Name Type Since Description
httpPort int - HTTP port to use. Defaults to 8080.
resourcesDirectory File - Directory for static resources, JSPs, etc.
webXml File - If web.xml needs to be picked up from another location, you can do that by specifying its location here.

Defaults to $resourceDirectory/WEB-INF/web.xml

Parameter Details

httpPort :

HTTP port to use. Defaults to 8080.

  • Type: int
  • Required: No

resourcesDirectory :

Directory for static resources, JSPs, etc.

  • Type: java.io.File
  • Required: No
  • Expression: ${basedir}/src/main/webapp

webXml :

If web.xml needs to be picked up from another location, you can do that by specifying its location here.

Defaults to $resourceDirectory/WEB-INF/web.xml

  • Type: java.io.File
  • Required: No
